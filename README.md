# AWS Cloud Architecture Project

##  Project Description
Designed a scalable and highly available cloud infrastructure using AWS services.

## 🛠 Services Used
- EC2
- Load Balancer
- Auto Scaling
- S3
- RDS
- CloudFront
- VPC

##  What I Did
- Created VPC with public and private subnets
- Configured load balancing and auto scaling
- Secured database in private subnet
- Implemented cloud storage using S3
